수정 내용
1. 라인업
- 청백전 선택 시에만 홈팀/원정팀, 주장/팀명 표시
- 연습경기/리그경기 선택 시 주장/팀명 숨김
- 청백전은 같은 날짜/시간에 홈팀/원정팀 라인업 2개 저장 가능
- 저장된 라인업 날짜 검색 추가

2. 기록지
- 경기 종류에 청백전 추가
- 청백전 선택 시 상대팀 입력칸 대신 상대팀 주장 선택 드롭다운 표시
- 경기종류+날짜+시간+상대팀 주장과 일치한 라인업을 타순에 자동 연동

3. 기록실
- 구단주만 보이는 선수 관리 탭 복구
- 기록지에 1회 이상 등장한 선수 닉네임순 표시
- 일반/용병/방출 설정 가능
- 용병/방출은 타율 순위에서 숨김

적용 방법: zip 안의 파일 전체를 기존 사이트 폴더에 덮어쓰기 후 GitHub에 업로드, Ctrl+F5 새로고침.
주의: 선수 관리 저장을 쓰려면 Firestore rules에 playerStatus 권한이 필요합니다.
